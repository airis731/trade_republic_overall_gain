# trade_republic_overall_gain
Here is some simple code to see what your overall gain is from Trade Republic transactions.

First you need to get the transactions from Trade Republic as a .csv file, it is, by this date 17.07.2026 atleast, available to be done directly from Trade Republic website.
Run the code and select the .csv file, no previous formating has to be done.

To know your overall current gain just add profits showed in the app and the "Total Welth Generated" that is given by the code.

Enjoy!
